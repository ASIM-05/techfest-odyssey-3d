# Techfest Odyssey 3D

An immersive 3D interactive Techfest website featuring scroll-based animations, interactive objects, and dynamic scene transitions.

## Live Demo
https://techfest-sphere-showcase.lovable.app
## Features
- 3D scroll animations
- Interactive 3D objects
- Dynamic transitions
- Responsive design

## Tech Stack
- React
- Three.js / React Three Fiber
- GSAP
- Vite

## Run Locally

```bash
npm install
npm run dev
```
